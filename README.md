## 自动构建
本项目配置了GitHub Actions自动构建：
- 推送到主分支时自动触发构建
- 构建结果将作为artifacts上传
- 可以直接从GitHub下载APK文件
## 本地构建
如需本地构建，请确保安装了：
- Python 3.10+
- Java 11+
- python-for-android
运行：
`ash
pip install -r requirements.txt
p4a apk --private src/main.py --requirements python3,kivy --name "APK项目" --package com.example.app --version 1.0.0
`
